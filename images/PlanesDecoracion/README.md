# Imágenes para Planes de Decoración

## Instrucciones para agregar las imágenes

Coloca en esta carpeta las siguientes imágenes de los planes de decoración:

### Archivos necesarios:

1. **plan-plata.webp** - Imagen del Plan Plata
   - Decoración con globos plateados
   - Estilo clásico

2. **plan-oro.webp** - Imagen del Plan Oro
   - Botella de vino/champagne
   - Arreglo floral elegante
   - Estilo preferido

3. **plan-luxury.webp** - Imagen del Plan Luxury
   - Jarrón con rosas
   - Pétalos decorativos
   - Estilo exclusivo

### Especificaciones recomendadas:

- **Formato**: WebP (para optimización)
- **Dimensiones**: 600x400px mínimo
- **Peso**: Menos de 200KB por imagen
- **Orientación**: Horizontal

### Alternativas si no tienes WebP:

También puedes usar:
- .jpg
- .png

Solo actualiza las referencias en el archivo HTML.

### Placeholder actual:

Mientras no tengas las imágenes, el sistema mostrará un placeholder con el texto del plan correspondiente.
